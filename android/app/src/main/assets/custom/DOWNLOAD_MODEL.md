# Download Whisper Model

Please download the **ggml-tiny.en.bin** model file and place it in the `assets` folder.

## Download Link

https://huggingface.co/ggerganov/whisper.cpp/resolve/main/ggml-tiny.en.bin

## Instructions

1. Click the link above to download the model file (~75MB)
2. Save it to: `c:\Users\HOME\Documents\VoiceChat\assets\ggml-tiny.en.bin`

The model file MUST be named exactly: **ggml-tiny.en.bin**
